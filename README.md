# blockchain_js

A minimal blockchain implementation written in JavaScript.  
This project is an educational exploration of how blockchains work internally—blocks, hashing, linking, and basic validation—without external libraries or frameworks.

---

## Features

- Simple `Block` and `Blockchain` classes
- SHA-256 hashing for block integrity
- Automatic linking between blocks
- Basic chain validation logic
- Minimal, readable codebase intended for learning
